# Bhakti — Professional Portfolio

A modern, responsive developer portfolio built with React, TypeScript, Vite, and Tailwind CSS.

## Sections

- **Hero** — Introduction and call-to-action
- **About** — Bio and highlights
- **Skills** — Tech stack by category
- **Projects** — Featured and other work
- **Experience** — Career timeline
- **Contact** — Email and social links

## Customize your content

Edit a single file: `src/data/portfolio.ts`

Update your name, email, social links, projects, experience, and skills. Replace `#` URLs with real links when ready.

## Run locally

```bash
cd portfolio
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build for production

```bash
npm run build
npm run preview
```

Deploy the `dist` folder to [Vercel](https://vercel.com), [Netlify](https://netlify.com), or GitHub Pages.

## Deploy to Vercel (recommended)

1. Push this folder to a GitHub repository.
2. Import the repo on [vercel.com](https://vercel.com).
3. Vercel auto-detects Vite — click Deploy.
